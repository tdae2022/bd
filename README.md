# Think Different Automation & Engineering Website

## 🏭 Project Overview
A professional industrial automation parts selling website for Bangladesh market, featuring products like PLC, HMI, Frequency Inverters, Analog Cards, Input/Output Cards, and PLC Modules from brands like Siemens, Mitsubishi, ABB, Allen Bradley, Schneider, Omron, Delta.

---

## ✅ Completed Features

### Pages
- **Home Page** (`index.html`) — Hero, brand bar, product categories, featured products, why-us, about, testimonials, CTA, footer
- **Products Catalog** (`products.html`) — Full listing with filters, search, sort, pagination, grid/list view
- **Contact Page** (`contact.html`) — Inquiry form, business hours, payment methods, map section

### Functionality
- 🛒 **Shopping Cart** — Add/remove items, update quantities, persistent via localStorage
- 💬 **Quote Request** — Cart modal with customer details form
- 🔍 **Product Filtering** — Filter by category, brand, price range, availability
- 🔎 **Search** — Real-time product search across name, brand, description, tags
- 📊 **Sort** — Sort by price (asc/desc), name A-Z
- 📄 **Pagination** — 12 products per page
- 📱 **Responsive Design** — Mobile, tablet, desktop
- 🌟 **Animated Counters** — Statistics animation on scroll
- ⬆️ **Scroll to Top** — Sticky scroll button
- 📋 **Contact Form** — With success feedback
- 🏷️ **URL Category Filtering** — `products.html?cat=plc` works directly
